# How to start

```
$ npm install

$ npm start

```
Troubleshoot
```
$ npm restart
```
When importing the project to an IDE check IDE for prompt to install npm (important)

or

```
$ npm install
$ nodemon server.js

$ npm start
```

Open Browser and insert URL:
```
localhost:3000
```

```

Google Chrome
MS Edge
Safari

Safari

```
